# Library Manager

## Overview
This project is a comprehensive Library Management System built using C#, and WPF. 
The system allows for efficient management of a library's various resources, including books, magazines, role management and staff tasks.

## Features
- Catalog management: Add, edit, or delete books
- User management: Manage roles of users
- Borrowing and returning: Easy-to-use UI for checking out and returning books
- Real-time updates: Changes are reflected immediately across all connected clients

## Technologies Used
- C#
- WPF (Windows Presentation Foundation)
